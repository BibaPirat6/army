# 🪖 Army Information System

Информационная система для автоматизации внутренних процессов военного комиссариата.

Проект разработан в качестве дипломной работы, но представляет собой полноценную информационную систему, решающую реальные задачи кадрового и организационного управления. Основная цель проекта — сократить количество ручной работы, централизовать хранение данных и упростить управление структурой организации, сотрудниками и ежедневными задачами.

---

# 📌 О проекте

Система предназначена для автоматизации работы военного комиссариата и позволяет вести организационную структуру, штатное расписание, сотрудников, назначения, календарь задач и формировать необходимые отчёты.

Проект ориентирован на использование во внутренней сети организации и поддерживает разграничение прав доступа между администраторами и сотрудниками.

# 🚀 Запуск проекта

## 1. Клонирование

```bash
git clone https://github.com/BibaPirat6/army.git
```

## 2. Установка зависимостей

```bash
composer install
npm install
```

## 3. Настройка окружения

```bash
cp .env.example .env
php artisan key:generate
```

Укажите параметры подключения к базе данных в файле `.env`.

---

## 4. Выполнение миграций

```bash
php artisan migrate
```

---

## 5. Запуск проекта

```bash
npm run dev

php artisan serve
```
